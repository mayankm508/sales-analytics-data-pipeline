# Sales & Supply Chain Analytics Data Pipeline

## 📌 Overview
This project demonstrates an end-to-end data analytics solution involving data extraction, transformation, and visualization. The goal is to analyze sales and supply chain data to generate actionable business insights.

## 🚀 Key Features
- Built ETL pipeline using Python and SQL
- Data cleaning and transformation of raw datasets
- Designed fact and dimension tables (data modeling)
- Developed KPI-based dashboards using Power BI
- Performed trend analysis on revenue, region, and product performance
- Used Excel for data validation and pivot analysis

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- SQL (MySQL)
- Power BI
- Excel (Pivot Tables, VLOOKUP, Charts)

## 📊 KPIs Tracked
- Monthly Revenue
- Regional Sales Performance
- Top Products
- Demand Trends
- Order Volume

## 🔍 Key Insights
- Identified top-performing regions contributing 40% of revenue
- Detected seasonal demand spikes
- Highlighted underperforming product categories

## 📁 Project Structure
- `data/` → Raw & cleaned datasets
- `sql/` → Queries for extraction & transformation
- `python/` → ETL scripts
- `dashboard/` → Power BI dashboard
- `excel/` → Data validation & analysis

## 📌 Conclusion
This project demonstrates strong skills in SQL, data pipelines, dashboarding, and analytics, aligned with real-world business use cases.
